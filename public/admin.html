Admin Control

Control panel for administrators to control the rollout of the matching proccess

set deadline for sign-ups

set new deadline for submissions

Notify Users

Notify users after the deadline has passed that their matches are ready

view data

view current users and matches

update languages
Update the list of languages offered

NOTE: languages must be separated by a single comma with no space, and formatted the way you intend users to view it
Example: English,Portuguese,Hiri Motu,Kalmyk

reset data

wipe current data for new submission period
